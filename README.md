# 🍕 NutriVision AI

An AI-powered Food Recognition and Nutrition Analysis Web Application built using TensorFlow, MobileNetV2, and Flask.

NutriVision AI enables users to upload food images and receive intelligent predictions along with estimated nutritional information such as calories, protein, carbohydrates, and fat content.

---

## 📌 Features

* Food Image Recognition using Deep Learning
* MobileNetV2 Pre-trained CNN Model
* Real-time Food Classification
* Nutrition Information Estimation
* Confidence Score Prediction
* Modern Flask-based Web Dashboard
* Image Upload and Preview Functionality
* Responsive User Interface

## ⚙️ How It Works

1. User uploads a food image.
2. The image is preprocessed and resized.
3. MobileNetV2 analyzes the image.
4. The model predicts the food category.
5. Confidence score is generated.
6. Nutritional information is displayed on the dashboard.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ShreejithShetty15/PRODIGY_ML_05.git
cd PRODIGY_ML_05
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
